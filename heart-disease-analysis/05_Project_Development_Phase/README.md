# 💻 05 – Project Development Phase

## 📖 Overview

The Project Development Phase is the implementation stage of the Heart Disease Analysis project. In this phase, the healthcare dataset was processed and analyzed using SQL, followed by the creation of an interactive dashboard in Tableau. The final dashboard was exported as an HTML file for web-based visualization.

This phase converts the project plan into a functional analytical solution that helps users explore heart disease patterns and risk factors through interactive visualizations.

---

# 🎯 Objectives

The objectives of this phase are:

- Prepare and organize the healthcare dataset.
- Perform SQL-based data analysis.
- Create meaningful analytical queries.
- Develop an interactive Tableau dashboard.
- Export the dashboard for web viewing.
- Organize all project development files.

---

# 📂 Project Files

## 📄 Heart_new2.csv

This is the primary dataset used in the project.

The dataset contains patient healthcare information such as:

- Age
- Gender
- Chest Pain Type
- Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- ECG Results
- Maximum Heart Rate
- Exercise Induced Angina
- ST Depression
- Heart Disease Status

This dataset serves as the foundation for the complete analysis.

---

## 📄 heart disease analysis.sql

This file contains all SQL queries developed during the project.

The queries were used for:

- Data exploration
- Filtering records
- Aggregating healthcare information
- Finding trends
- Identifying risk factors
- Preparing data for visualization

The SQL analysis provides meaningful insights from the healthcare dataset.

---

## 📄 heart disease analysis.twbx

This is the Tableau Packaged Workbook.

It contains:

- Interactive Dashboard
- Charts
- Filters
- KPIs
- Visual Analysis
- Dashboard Layout

The workbook combines SQL analysis with visual analytics to help users understand heart disease trends effectively.

---

## 📄 heart_disease_dashboard.html

This file is the exported interactive dashboard.

It allows users to explore the dashboard through a web browser without opening Tableau Desktop.

The dashboard includes:

- KPI Cards
- Interactive Filters
- Charts
- Risk Analysis
- Demographic Analysis
- Medical Factor Analysis

---

## 📁 heart-disease-dashboard

This folder contains the files required to display the project dashboard through a web interface.

It includes:

- HTML pages
- CSS styling
- JavaScript files
- Project resources

This folder helps present the dashboard in an organized and user-friendly format.
