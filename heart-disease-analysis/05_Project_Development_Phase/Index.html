Reading the signals behind heart disease risk

An end-to-end analysis of demographic, medical, and lifestyle factors associated with heart disease — from raw data to an interactive Tableau dashboard, built on a SQL + Python + Flask pipeline.

01 — Overview

What this project does

This project analyzes patient-level health data to identify which demographic, medical, and lifestyle factors correlate most strongly with heart disease. Data was cleaned and modeled in SQL, explored statistically in Python, and surfaced through an interactive Tableau dashboard served via a lightweight Flask app — letting a non-technical reviewer explore the same patterns a data scientist would find in a notebook.

03 — Methodology

How it was built

The pipeline moves from raw data to an interactive dashboard in four stages:

STEP 1

Clean & structure

Raw clinical data cleaned and normalized in SQL; missing values and outliers handled.

STEP 2

Explore & model

Statistical analysis and feature importance ranking to surface the strongest risk indicators.

STEP 3

Visualize

Findings built into an interactive Tableau workbook for exploration by feature and segment.

STEP 4

Serve

Dashboard embedded and served through a lightweight Flask application.
