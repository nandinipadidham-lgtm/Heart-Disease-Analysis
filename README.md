# 📖 About the Project

Heart Disease Analysis is a comprehensive healthcare data analytics project developed to analyze patient health records and identify key factors associated with heart disease.

The project demonstrates a complete analytics workflow, beginning with problem identification and requirement gathering, followed by solution design, project planning, data analysis, visualization, performance validation, and final documentation.

Using SQL for data analysis and Tableau for interactive visualization, the project transforms raw healthcare data into meaningful insights that can assist healthcare professionals, researchers, and decision-makers in understanding disease trends and patient characteristics.

The repository has been organized into different project phases to clearly demonstrate every stage of the software and analytics development lifecycle.

---

# 🎯 Project Objectives

- Analyze heart disease patient data to discover meaningful healthcare insights.
- Identify major risk factors associated with heart disease.
- Perform structured SQL-based data analysis.
- Develop interactive Tableau dashboards for data visualization.
- Present healthcare trends using KPIs and visual analytics.
- Document every stage of the project lifecycle professionally.
- Demonstrate practical knowledge of healthcare data analytics.
